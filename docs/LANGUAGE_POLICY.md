# Política de idioma

Política oficial del proyecto Lumind Intelligence API. Referenciada desde [AGENTS.md](../AGENTS.md).

---

## Reglas

| Ámbito | Idioma | Ejemplos |
|--------|--------|----------|
| **Código** | Inglés | Clases, métodos, variables, endpoints, DTOs, nombres de paquetes, commits técnicos |
| **Documentación interna** | Español | `/docs`, `AGENTS.md`, ADRs, specs, sprints, domain model, DoD |
| **README público** | Inglés (Sprint 9) | Versión definitiva para GitHub y portfolio |

---

## Reglas derivadas

- Los identificadores de código **nunca** se traducen al español.
- `AGENTS.md` siempre se mantiene en español.
- El README actual es provisional en español; se traduce al inglés en Sprint 9.
- No mantener versiones bilingües permanentes del mismo documento.
- Los documentos en `/docs` que aún estén en inglés se traducen **progresivamente**, cuando se modifiquen, no de forma masiva.

---

## Para agentes IA

- Escribe código y APIs en inglés.
- Escribe documentación interna y respuestas de planificación en español.
- Al editar un documento interno en inglés, tradúcelo al español como parte del mismo cambio.
